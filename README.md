# Customer Segmentation using Machine Learning

## Overview

This project applies **Machine Learning clustering techniques** to segment mall customers based on their purchasing behavior, annual income, and spending patterns.

The goal is to help businesses understand different customer groups and improve marketing strategies through data-driven insights.

---

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Customer Segmentation using K-Means Clustering
* Data Visualization with Matplotlib & Seaborn
* PCA-based Cluster Visualization
* 3D Customer Segmentation Visualization
* Business Insights & Recommendations
* Interactive Streamlit Web Application
* Power BI Dashboard

---

Mall Customers Dataset

---

## Machine Learning Workflow

### 1. Data Preprocessing

* Loaded customer dataset
* Checked missing values
* Performed statistical analysis
* Selected important features

### 2. Data Visualization

* Age distribution analysis
* Income vs Spending Score analysis
* Cluster visualization plots

### 3. Feature Scaling

Used StandardScaler to normalize features before clustering.

### 4. K-Means Clustering

Applied K-Means clustering algorithm to identify customer groups.

### 5. Elbow Method

Used the Elbow Method to determine the optimal number of clusters.

### 6. Business Insights

Generated meaningful customer behavior insights and marketing strategies.

## Business Insights

### High Income + High Spending

* Premium customers
* Ideal for luxury product marketing

### High Income + Low Spending

* Potential customers
* Can be targeted with personalized campaigns

### Low Income + High Spending

* Responsive to discounts and promotional offers

### Low Income + Low Spending

* Budget-conscious customers
